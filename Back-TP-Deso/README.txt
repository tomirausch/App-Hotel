	
	¿Como correr el proyecto?
	
	NetBeans:
	
		1) Abrir el proyecto (File --> Open Proyect)
		2) Click Derecho en el Proyecto --> "Clean and Build"
		3) Una vez haya finalizado el "Clean and Build" --> Run Proyect (Boton de Play Verde)
		4) Luego de que haya finalizado el Run Proyect ingresar a:
		
			http://localhost:8080/
			
		5) Probar el CU 09 - Dar de Alta Huesped
		
	VS Code:
	
		1) Abrir el proyecto (File --> Open File)
		2) RUN --> Start Debugging --> Java
		3) Una vez haya finalizado todo el proceso ingresar a:
		
			http://localhost:8080/
			
		5) Probar el CU 09 - Dar de Alta Huesped
